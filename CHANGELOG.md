## [0.0.1] - 2026-05-22

Initial scaffold of the maxvision-skills curated catalog repository.

### Added

- Plugin manifest (`.claude-plugin/plugin.json`).
- Marketplace entry (`.claude-plugin/marketplace.json`).
- `skills/` placeholder directory.
- README, CONTRIBUTING, LICENSE.
- CI workflow scaffolding (markdown lint + JSON syntax).
- Registered as Tier 2 source in the maxvision plugin catalog at
  [`skills/discover-skill/references/skill-sources.json`](https://github.com/produtoramaxvision/maxvision/blob/main/skills/discover-skill/references/skill-sources.json).

### Not yet implemented

- Skill submission auto-PR pipeline (planned for maxvision plugin v0.1.1).
- Branch protection rules (admin action, run separately).
- PR/issue templates.
